# HR Attrition Analysis

## Project Overview
Analyzed 1,470 employee records to identify key factors 
driving employee attrition using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI

## Dataset
IBM HR Employee Attrition Dataset (1,470 records, 35 columns)

## Key Findings
1. Overall attrition rate is 16%
2. Sales department has highest attrition
3. Employees earning less leave more often
4. Age group 25-35 has highest attrition
5. Poor work-life balance strongly linked to attrition

- HR_attrition_dashboard.pbix - Power BI Dashboard
